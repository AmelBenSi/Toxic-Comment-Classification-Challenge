# Toxic-Comment-Classification-Challenge
Multi-label toxic comment classification (Kaggle) — an end-to-end NLP notebook (imported from Google Colab) covering EDA, text cleaning with spaCy, TF-IDF + spaCy vector embeddings, and multi-output ML models (LogReg/SGD/RF/MLP) with Accuracy &amp; ROC-AUC evaluation and Kaggle-ready predictions.
